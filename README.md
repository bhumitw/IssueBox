# Issue Box

A relationship wellness app enabling safe communication, appreciation, and growth tracking for couples.

## Local Setup

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Environment Setup**
    Create a `.env` file in the root directory and add your Google Gemini API Key:
    ```
    API_KEY=your_gemini_api_key_here
    ```

3.  **Run Locally**
    ```bash
    npm run dev
    ```
    Open the URL provided (usually `http://localhost:5173`).

## Features
- **Issue Box**: Write private vents, refine them using AI to remove aggression, and share them only when ready.
- **Appreciation Box**: Share gratitude and little moments of love.
- **Mediator**: An AI chatbot acting as a neutral third party to help resolve disputes using context from your relationship history.
- **Growth Map**: Gamified relationship timeline tracking memories, dates, and trips.
- **Dashboard**: Control your "Ready to Receive" state to protect your peace.
